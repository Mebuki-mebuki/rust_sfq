# RSFQlib (Extracted files)

This directory contains a subset of files extracted from the
RSFQ cell library developed by

Lieze Schindler and Tessa Hall,
Stellenbosch University

Version: 3.0  
Release date: 21 March 2023

Only the files required for simulation in RustSFQ are included.

Additional files such as `all.v` and `all.cir` were added to simplify
including the entire library.

The original license is included in `LICENSE`.
